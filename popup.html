VS ModDB Blocker

Version 5.6 Allows users of the Vintage Story Moddb to use advanced filters like Block tags, Block authors, Block mods, Block keywords in description, Allow keywords and lets you either make them less visable or hide them completely.

Block list

Exact match only. Example: l33tmaan
Examples: Cantina Atlas, 32289, or 5444.
Plain AI = contains anywhere. "AI" or word:AI = free-standing word only.
Exact tag match only. Example: Tweak.

Keyword allow list

Example: block "AI", but allow "no AI". Same rule syntax as blocked keywords.
Debug mode keeps the old behavior so you can watch cards get checked and hidden one by one.
