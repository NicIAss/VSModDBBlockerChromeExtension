# VSModDBBlockerChromeExtension
Allows users of the Vintage Story Moddb to use advanced filters like Block tags, Block authors, Block mods, Block keywords in description, Allow keywords and lets you either make them less visable or hide them completely.

## What does this extension do?

VSModDBBlockerChromeExtension helps users customize their browsing experience on the Vintage Story ModDB by filtering mod cards based on personal preferences until the page itself implements it.

You can use it to:

- Block specific tags
- Block specific authors
- Block specific mods
- Block keywords found in mod descriptions
- Allow specific keywords
- Make blocked mods less visible
- Hide blocked mods completely

## How to use it

1. Install the extension in Chrome or another Chromium-based browser.
2. Open the Vintage Story ModDB website.
3. Open the extension settings.
4. Add the tags, authors, mods, or keywords you want to block.
5. Choose whether blocked mods should be only less visible or fully hidden.
6. Save your changes at the bottom and refresh the ModDB page and the extension will apply your filters.
